Paigo Web Admin

Role: Full-stack developer - Entire site
Company: Own Customer

Description

Paigo is a financial institution. This project was about a web app built to administrate customers, granted loans and other products.

I built this app from scratch to help them administer their business. During this project I was also contributing to find the best model for the problem they had.

We used Scrum to manage the project. Before every review I pushed a docker image and we used it as the output of the sprint.

Technologies

Node.js, MongoDB, Express.js, Angular 6, Docker, Javascript, HTML, CSS, Bootstrap.

Unfortunately this project has not public access as it's an internal web for the company.

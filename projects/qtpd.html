¿Qué tenés para dar?

Role: Co-Founder & Full-stack developer

Description

As a co-founder, I have touched all the areas of the project. Our main goals are

  • To help people find where they can bring their donations
  • To help NGOs receive more donations by being visible in the map
  • To use the latest technologies so we can keep learning with a real-world app.
This platform is open source, we are currently updating the stack to use MERN and we will open it to any developer who wants to collaborate with the society! The technologies of this project have been evolving with the time.

Technologies

React.js, Node.js, MongoDB, Express.js, Angular 8, Docker, Javascript, HTML, SCSS, Bootstrap 4, Python, Django.
